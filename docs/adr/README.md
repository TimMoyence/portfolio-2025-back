# ADR Backend

Les ADR gardent une trace concise et datee des choix d'architecture, de gouvernance et de compromis techniques.

## Regles

- Nommer les fichiers `NNNN-titre-court.md`
- Garder un seul sujet par ADR
- Mettre a jour le statut si une decision est remplacee
- Referencer l'ADR depuis la doc de gouvernance ou le README si le sujet impacte l'onboarding
